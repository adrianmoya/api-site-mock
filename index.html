Compute

Servers

GET
/servers
list all servers in summary form
detail
ParameterDescription
image (optional) servers booted from a specific image - (see /images)
flavor (optional) description of parameter
name (optional) description of parameter
status (optional) lists servers in a given state, where those states can be one of: ACTIVE, BUILD, REBUILD, SUSPENDED, RESIZE, VERIFY_RESIZE, PASSWORD, REBOOT, HARD_REBOOT, DELETED, UNKNOWN, and ERROR
changes-since (optional) lists the changes of servers since the timestamp specified (provided in ISO8601 format)

Example Request

GET
/servers?status=ACTIVE
